Edit `globals.css`:

```css

  from-[#YOUR_COLOR] via-[#YOUR_COLOR] to-[#YOUR_COLOR]
```

### Adjust Animations

Edit animation durations in `page.tsx`:

```tsx
<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.8 }} // Change this
>
```
